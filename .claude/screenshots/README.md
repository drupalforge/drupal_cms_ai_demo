# Screenshots for AI

Place screenshots here for AI agents to analyze.

## Usage

When you need Claude to look at something visual:
1. Take a screenshot
2. Save it to this directory
3. Reference it in your message or let Claude read it

## Naming Suggestions

- `issue-3512345-bug.png` - Screenshot related to a specific issue
- `ui-problem.png` - General UI issue
- `error-message.png` - Error screenshot
- `before.png` / `after.png` - Comparison screenshots
